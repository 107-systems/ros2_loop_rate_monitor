#### How-to-build
```bash
cd $COLCON_WS/src
git clone https://github.com/107-systems/ros2_loop_rate_monitor
cd $COLCON_WS
source /opt/ros/humble/setup.bash
colcon build --packages-select ros2_loop_rate_monitor
```
